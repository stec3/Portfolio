# Todo Manager (CLI)

A simple command-line todo manager written in Python.

## Features
- Add a task
- List tasks
- Remove a task
- Exit

## How to run
```bash
python3 main.py
```

## Concepts used
- Lists and dictionaries
- Functions
- Input/output
- Control flow

## Future improvements
- Persist tasks to a file
- Mark tasks as completed
- Improve input validation
- Refactor code structure
